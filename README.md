# Signal and noise
Creative Coding Lab for 2026

A collection of creative coding experiments, simulations, generative art sketches, and interactive systems built with Processing.

This repository serves as:
- a learning space,
- a visual sketchbook,
- a reusable code library,
- and a long-term portfolio of creative coding projects.

The goal is to gradually explore:
- animation,
- particle systems,
- procedural generation,
- physics simulations,
- interaction,
- audio visualization,
- and eventually hardware integration with Arduino.

---

## Repository Structure

```text
.
├── sketches/   # Individual Processing sketches and experiments
├── library/    # Reusable classes, utilities, and systems
├── exports/    # GIFs, videos, and rendered outputs
└── README.md
```

---

## Project Philosophy

This repository prioritizes:
- experimentation over perfection,
- iteration over polish,
- and consistency over complexity.

Some sketches may be simple studies, while others may evolve into larger interactive systems.
