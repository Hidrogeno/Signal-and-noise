# 004 Gravity Wells:

![Gravity Wells Preview](../../../exports/gifs/004_GravityWells.gif)

In this sketch we reuse the particle and force concepts and add attractors as well as repulsors with the mouse, we can remove them in a blast by pressing 'c' on the keyboard.

Since this gif is interactive, the gif alone doesn't make it justice, a more detailed video can be seen in the 'videos' folder.
## Concepts:
- Vectors
- Velocity
- Fading trails
- Acceleration
- Forces
- Mouse/keyboard interaction

## Controls:

- An attractor will be placed when right clicking on the screen
- A repulsor will be placed when left clicking
- Pressing 'c' on the keyboard will remove attractors/repulsors in a flash
